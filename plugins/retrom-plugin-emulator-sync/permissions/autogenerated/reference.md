## Default Permission

Default permissions for the emulator sync plugin

#### This default permission set includes the following:

- `allow-ensure-emulator-synced`
- `allow-get-emulator-sync-status`
- `allow-get-emulator-sync-index`
- `allow-subscribe-to-emulator-sync-updates`
- `allow-unsubscribe-from-emulator-sync-updates`
- `allow-abort-emulator-sync`
- `allow-open-emulator-cache-dir`
- `allow-analyze-emulator-user-data`
